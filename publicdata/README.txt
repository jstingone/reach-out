This folder contains programming code and data associated with public data used within the reach-out project.

Specific folders/files are:

* NYCCAS_exposure: Folder containing programming code and data associated with air pollution exposure data derived from NYCCAS data

* TestingRate_sensitivity: Folder containing programming code and data associated with COVID19 testing data by NYC ZIP Code Areas, used for sensitivity analyses

* DD_nevi_zip_final.xlsx: EXCEL Workbook of the Data Dictionary Corresponding to the Neighborhood Environmental Vulnerability Index; Data Available at the NEVI Repository github.com/jstingone/nevi
