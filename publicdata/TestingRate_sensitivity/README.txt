This folder contains programming code to process public COVID19 testing data originally obtained from NYC OpenData and stratify NYC Zip Code Tabulation areas into tertiles of levels of testing. It includes a final dataset and data dictionary.

Specific files are:

* TestingRate.RMD: R Markdown file that processes publicly available COVID19 testing data to stratify ZIP Code Tabulation Areas by amount of testing, accounting for population size

* TestRate_Tertiles.csv: CSV file containing the weekly testing rates and overall assigned tertiles for each NYC ZCTA

* DD_TestRate_Tertiles.xlsx: EXCEL Workbook of the Data Dictionary Corresponding to Data file TestRate_Tertiles.csv
