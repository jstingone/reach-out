This folder contains programming code related to data preparation and data derived from publicly available sources that can be shared. It includes 3 folders:
 
* NYCCAS_exposure: code to construct exposure metrics for NYC ZIP Codes using public NYCCAS data from NYC OpenData. It includes final dataset and data dictionary for air pollution data. It also includes data dictionary for NEVI data;
* inpatientpop: code to construct the inpatient study population within INSIGHT, clean data and construct derived variables;
* EDpop: code to construct the emergency department study population within INSIGHT, clean data and construct derived variables.
